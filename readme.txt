== Description ==

WP Send is the simplest way to send your files around the world. Share large files with configurable filesizes and expiry dates.

== Installation ==

1. Upload the wp-send folder to your '/wp-content/plugins/' directory
2. Activate the plugin via the Plugins menu in WordPress
3. A new WP Send menu will appear in WordPress with options, and general help
